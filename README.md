# IWeather

IWeather is a simple and intuitive mobile application that provides accurate weather forecasts for any city. Just enter the desired city name to access information about current temperature, high and low temperatures, humidity, wind speed, and forecasts for the upcoming days.

## Lessons Learned

**Unit testing** is a fundamental aspect of software development aimed at ensuring the reliability and functionality of individual components within an application. In our recent learning session, we embarked on creating a robust set of unit tests for our application, aiming to bolster its stability and performance.
